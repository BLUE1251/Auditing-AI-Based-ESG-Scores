# Auditing AI-Based ESG Scores

## Overview

This project audits AI-based ESG scoring systems by evaluating their robustness, explainability, fairness, and predictive performance using ESG ratings and sustainability reports. 

It also identifies potential greenwashing risks through disclosure-score discrepancy analysis to support transparent and responsible ESG governance.

## Research Questions

- RQ1: Robustness Analysis
- RQ2: Explainability Analysis
- RQ3: Fairness Audit
- RQ4: ESG Prediction from Sustainability Reports
- RQ5:Greenwashing Detection
